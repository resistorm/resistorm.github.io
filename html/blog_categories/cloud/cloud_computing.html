Cloud Computing

Het ‘werken in de cloud’ duidt op de bestanden die niet op je computer staan opgeslagen, maar op een locatie waartoe je toegang hebt via het Internet of een internet. De voordelen hiervan zijn de opslagcapaciteit, welke on-demand verhoogd of verlaag kan worden, en de mogelijkheid vanaf verschillende apparaten bij je bestanden te kunnen. Hierdoor hoeft er bijvoorbeeld niet geïnvesteerd te worden in opslag hardware. Cloud computing is het vervolg op deze technologie die het mogelijk maakt om intensieve applicaties te gebruiken zonder te hoeven investeren in krachtige computers.

Cloud computing wordt door het nationale instituut voor standaarden in de verenigde staten gedefinieerd als:

Een model voor het mogelijk maken van alomtegenwoordige, gemakkelijke, on-demand netwerk toegang tot een gedeelde pool van computing resources die met minimale managementinspanning of interactie met een dienstverlener, zeer snel kunnen worden aangeleverd en vrijgemaakt.

Cloud computing is dut een model dat het mogelijk maakt om de computing resources te gebruiken die nodig hebt. Computing resources zijn de fysieke componenten in een computer. Cloud computing is dus een model waarmee hardware capaciteiten kunnen worden gebruikt, zonder hierin te hoeven investeren.

Velen zijn gewend IT te ervaren als een wegversperring. Dit heeft alles te maken met het feit dat bij ieder nieuw iets, eerst dure plannings-, bemiddelings- en implementatie cyclussen doorlopen moeten worden. Cloud computing echter, mag gezien worden als het tegenovergestelde hiervan, als een enabler.
